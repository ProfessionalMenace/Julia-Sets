# Julia-Sets
Julia sets is a simple CPU rendering project in C++ using GLFW + OpenGL.

# Planned Features
- [x] GPU rendering
- [x] Proper GUI
- [ ] Support for other fractals
- [x] Iteration based coloring
- [ ] Output
